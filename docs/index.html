Research initiative

Reasoning systems should govern disagreement, not hide it.

Sovereign Epistemic Agent is a research direction for machine reasoning architectures that stage competing internal accounts, critique them against structure and memory, and decide when selection is justified, when resampling is needed, and when abstention is the honest outcome.

What this is

An umbrella project with one concrete subsystem already running

Sovereign Epistemic Agent

The umbrella identity and long-range direction. It names a class of reasoning systems that treat internal disagreement, failure reuse, and deliberative restraint as first-order design concerns.

Epistemic Tribunal

The implemented experimental stack in this repository. It operates on ARC-like tasks by generating competing traces, extracting invariants, critiquing candidates, measuring uncertainty, adjudicating outcomes, and recording failures.

Implemented stack

The Epistemic Tribunal turns private model guesses into inspectable procedure.

Generator bank

Five strategies produce competing candidate traces instead of a single unchallenged output.

02

Invariant extractor

Task-level structural constraints are inferred from training examples and activated by confidence.

03

Trace critic

Candidates are scored for consistency, rule coherence, morphology, failure similarity, and compliance.

04

Uncertainty analyzer

Entropy, margin, coalition mass, and disagreement rate quantify how fractured the pool really is.

05

Tribunal aggregator

Weighted signals lead to select, resample, or abstain rather than forced confidence theater.

06

Failure ledger

SQLite captures traces, decisions, invariant violations, and failures as reusable memory.

Design philosophy

The project is built around disciplined refusal.

Competing hypotheses

The central object is not the answer but the governed conflict between plausible accounts of the task.

Structural accountability

Invariants and critics force candidate traces to answer to shape, symmetry, and prior evidence.

Memory with consequence

Failures are written down in a structured form so the system can stop repeating them blindly.

Abstention as competence

Low confidence is not hidden. The tribunal can resample or abstain when selection is not defensible.

Status map

Long-range direction
  • Strange Loop memory during reasoning
  • Richer co-agency or operator-mind architectures
  • Honest handling of undecidable or structurally unresolved cases
